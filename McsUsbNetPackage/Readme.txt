README for McsNetUsb.dll
========================

This zip file contains the McsUsbNet.dll for accessing MCS devices directly.

It does not contain any support for reading and writing specific MCS file formats
as the MCD file format from MC_Rack or the STM file format from MC_Stimulus

In the folder "Examples" you find simple example project for some device classes.
In the folder "Doc" you find preliminary documentation for the library
Note: "McsUsbNet.*" describes the complete Dll.
"McsUsbNet_for_STG.*" is a subset describing the access of MCS STG devices.
"McsUsbNet_for_DAQ.*" is a subset describing the access of MCS data aquistion devices (MC_Card and USB-ME(A))
